pub enum StreamMaxlen {
    Equals(usize),
    Approx(usize),
}

Utility enum for passing MAXLEN [= or ~] [COUNT] arguments into StreamCommands. The enum value represents the count.

Variants

Equals(usize)

Match an exact count

Approx(usize)

Match an approximate count
